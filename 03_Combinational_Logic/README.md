# Combinational Logic

This section contains HDLBits exercises and additional RTL implementations demonstrating fundamental combinational digital design concepts in Verilog.

## Topics Covered

- Boolean logic design
- Karnaugh map implementation
- MUX-based logic design
- Multiplexers (MUX)
- Decoders
- Comparators
- Parameterized designs
- Priority encoders
- Arithmetic circuits
- Full adders
- Ripple carry adders
- Population count
- Truth table implementation
- Conditional operators
- always blocks
- case statements
